Push Stock

This group project was developed as a Senior Capstone Project during our teams last semester at Baylor University. This project was provided to us by Pariveda Solutions. The goal of this project was to integrate all of the knowledge learned throughout college and apply it in a situation that resembled situations found in industry. We were given a fictional problem and we had 15 weeks to implement a solution for such a problem. The solutions was an IOT involving a raspberry pi, BTLE Flic button, and a CentOS server. We used a MEAN stack which was comprised of: MONGO DB, ExpressJS, Angular2 (ionic for mobile application), and NodeJS
